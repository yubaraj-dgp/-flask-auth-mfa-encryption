# 🔐 Secure MFA Encryption App

This project is a Python-based web application built using Flask. It demonstrates a basic security system combining authentication, multi-factor authentication (MFA), and encryption.

## 🚀 Features
- User login system (username & password)
- OTP-based Multi-Factor Authentication (MFA)
- Caesar Cipher encryption & decryption
- Secure flow: Login → OTP → Dashboard

## 🛠️ Technologies Used
- Python
- Flask
- HTML (for frontend)

## ⚠️ Note
- OTP is currently displayed in the terminal (for testing purposes)
- Caesar Cipher is used for learning purposes and is not secure for real-world use

## 📌 Future Improvements
- Email/SMS-based OTP system
- Database integration (SQLite/MySQL)
- Strong encryption like AES
